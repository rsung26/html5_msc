#General Assembly WDI Final Project
##GA-NLP: HTML5 Marine Split Challenge

Check out the site here: http://html5-msc.herokuapp.com/

###About this Project

This HTML5 game uses the Phaser framework to build an arcade style micro challenge map. Based on many ideas from arcade map Marine Split Challenge and Starcraft 2. Runs best on Google Chrome.

###Getting Started

To start playing, feel free to visit the website. This game was built using Phaser 1.1.6.

###Technolgies Used
* Phaser HTML5 Game Framework
* Ruby on Rails
* Twitter Bootstrap
